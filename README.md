# 華懋集團物業地圖 POC

## 項目簡介

這是一個展示**華懋集團（Chinachem Group）**旗下物業的互動地圖網站 POC（概念驗證）。

## 功能特點

- 🗺️ **互動地圖**：使用 Leaflet.js 顯示香港地圖
- 🏢 **物業分類**：
  - 商業辦公（藍色）
  - 零售商場（綠色）
  - 住宅項目（紅色）
  - 酒店服務（紫色）
  - 其他設施（灰色）
- 🔍 **搜尋功能**：可按物業名稱或地址搜尋
- 📋 **側邊欄列表**：顯示所有物業，點擊可定位到地圖位置
- 📱 **響應式設計**：適配桌面和移動設備

## 技術棧

- **前端**：HTML5, CSS3, JavaScript (ES6+)
- **地圖庫**：Leaflet.js 1.9.4
- **字體**：Google Fonts (Noto Sans TC)
- **圖標**：Emoji

## 使用方法

### 本地運行

1. 打開 `index.html` 文件：
   ```bash
   # 使用瀏覽器直接打開
   open index.html
   
   # 或使用本地服務器
   cd chinachem-property-map
   python3 -m http.server 8000
   # 然後訪問 http://localhost:8000
   ```

2. 即可在地圖上查看所有物業位置

### 在網上部署

將整個文件夾上傳到任何靜態網站托管服務：
- GitHub Pages
- Vercel
- Netlify
- AWS S3
